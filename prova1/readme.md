## Como executar 
1) Clone o repositório
2) Execute os comandos abaixo
   va até a pasta de pub
```
go run publisher.go
```
va até a pasta de sub
```
go run subscriber.go
```
Para testar
```
go mod tidy
```
```
go test
````
